# OpenModels

Open infrastructure for discovering and comparing LLM models.
OpenModels is a community-driven project focused on normalizing the modern AI inference ecosystem — models, providers, pricing, rate limits, compatibility and capabilities.

We aim to become the open source of truth for:

- LLM models
- inference providers
- OpenAI-compatible APIs
- pricing and rate limits
- SDK compatibility
- inference capabilities
- provider implementations
